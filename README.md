# name-that-color-cn
颜色命名小工具， 帮助程序员为颜色变量起一个更好理解的名字

## 使用方法
```javascript
let myColor = new Color('bd9b64');
myColor.getName(); // '昏黄'

myColor = new Color('b99459');
myColor.getName(); // '昏黄'
```
